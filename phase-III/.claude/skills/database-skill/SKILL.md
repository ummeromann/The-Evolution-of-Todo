Skill: Database Design & Management

Description:
Ability to design, structure, and manage databases including schema design, table creation, relationships, and migrations for scalable and maintainable applications.

Scope of Skill:

Database schema design & normalization

Creating tables, columns, and relationships

Writing and managing migrations

Indexing for performance

Handling constraints (primary keys, foreign keys, unique, etc.)

Managing data consistency & integrity

Working with SQL & NoSQL databases

Tech Stack:
PostgreSQL, MySQL, MongoDB, Prisma, SQL, TypeScript, Python

Outcome:
Can design clean, scalable, and production-ready databases for web apps, SaaS platforms, dashboards, and AI systems.
