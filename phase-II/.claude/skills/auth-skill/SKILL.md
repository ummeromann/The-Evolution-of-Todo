Skill Name:
Auth Engineering (Signup, Signin, JWT, Better Auth)

Skill Description:
Ability to design, implement, and secure complete authentication and authorization systems including user registration, login, password hashing, JWT token handling, and Better Auth integration in modern web applications.

Skill Objective:
To build secure, scalable, and production-ready authentication systems that follow industry best practices and protect user data.

Core Competencies (What this skill includes)

1. User Registration (Signup)
Input validation
Email/username uniqueness checks
Secure user creation flow

2. User Login (Signin)
Credential verification
Error handling (invalid email/password)
Session / token issuance

3. Password Security
Hashing with bcrypt/argon2
Salting strategies
Password strength enforcement

4. JWT Authentication
Access token generation
Refresh token flow
Token verification & expiration handling 

5. Route Protection
Middleware / guards
Protected APIs & pages
Role-based access control (RBAC)

6. Better Auth Integration
Setup & configuration
Provider-based auth (email, OAuth, etc.)
Session handling

7. Security Best Practices
OWASP guidelines
Rate limiting
Brute-force protection
Secure cookie handling
